# Cover-Art

Lege hier `cover.png` ab (der Dateiname aus `config.yaml` -> `podcast.cover_image`).

Apple-Podcasts-Anforderungen an das Cover:
- Quadratisch, **1400 x 1400** bis **3000 x 3000** px
- JPG oder PNG, RGB-Farbraum
- Klar erkennbar auch als kleines Thumbnail

Tipp: Du kannst das Cover selbst mit einem Bildgenerator (z. B. OpenAI Images)
erzeugen. Ohne gültiges Cover akzeptiert Apple Podcasts den Feed nicht.
